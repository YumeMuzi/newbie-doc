# 新人系列群可能碰到的问题

*原作者来自于 CYCLC。*

## 简单排查 osu 卡顿

### 台式电脑

1. 检查后台是否运行过多软件（例如 360），并将动态壁纸设置为在应用全屏时自动停止。
   - 除了部分特殊软件，大多数软件基本上不需要开机自启动并静默运行。
   - 在 Windows 设置中，搜索系统设置 -> 启动应用，关闭你认识的三方软件的自启动。
2. 检查 CPU 和显卡温度是否因过热导致降频。
3. Stable 版本设置：开启无限制帧数和全屏模式。
4. Lazer 版本设置：将 update 设置为 1000 或 960 以保证操作手感，draw 的帧数保证稳定超过刷新率即可。
   - 按 Ctrl+F11 即可在右下角看到 update 和 draw 的信息，再多按几次即可隐藏。
   - 调整设置 -> 图像 -> 帧数限制 -> 4x refresh rate（或其他满足以上条件的档位）。
5. N 卡用户设置：参照差异教程调整 NVIDIA 控制面板设置（必须关闭 G-SYNC）。

### 笔记本电脑

1. Stable 玩家：关闭兼容模式，开启无限制帧数和全屏模式。
2. Lazer 玩家：同上。
3. 确认笔记本类型（游戏本或轻薄本）：

#### 游戏本

- 检查后台程序和动态壁纸。
- 若为近几年的机型，在品牌控制台中开启**独显直连**功能，并切换至**最高性能**模式。
   - 独显直连：例如华硕的Armoury Crate、联想的Legion Zone
   - 最高性能：例如华硕的增强模式、联想的野兽模式。
- 检查 CPU 和显卡温度是否过高，若温度过高，建议更换相变片（如霍尼韦尔7950）并清灰。
   - 可以用手靠近笔记本电脑的出风口来感受风量（注意烫伤！），若出风量较小，则需要考虑清灰。可以联系相应的品牌售后点。不推荐自己拆开清灰（后果自负）。
- 游戏本基本上都是 NVIDIA 显卡（右下角的后台应用可见 NVIDIA 设置的小图标），需按上述方法调整控制面板并关闭 G\-SYNC。

#### 轻薄本

- 通常仅配备核显或无直连功能的独显，帧数较低。
- 建议优先更新核显驱动，检查温度和后台程序。
  - Intel 核显驱动：可通过必应搜索处理器型号后下载。
  - AMD 核显驱动：需前往官网驱动界面搜索处理器型号。
- 建议全新安装（卸载旧驱动后再安装）。

## 音频延迟

1. 如果遇到准确率很难提高的问题，先检查是否使用外放音箱或笔记本内置喇叭，而不是有线耳机。
2. 进一步确认耳机是否为 3.5mm 圆孔耳机：
   - 如非 3.5mm 耳机，建议换成 3.5mm 耳机并直接插在电脑上。
   - 外放、蓝牙、USB、2.4G 无线、小尾巴声卡及 USB 声卡都会带来额外的延迟。
3. 使用 3.5mm 耳机后，需修改音频驱动（大多数电脑会被微软自动安装品牌音频驱动，延迟非常高）：
   - 可直接看视频（[BV1DKmFBNE3C](https://www.bilibili.com/video/BV1DKmFBNE3C)）。
   - 如果改驱动后没有声音，回退驱动即可。
   - 没声音时，也可以使用显示器的 3.5mm 接口，并参照上面的视频修改显示器 3.5mm 接口的驱动（仅要改的驱动名字不同）。

还有其他问题直接问CYCLC。
